# 草包 iOS 客户端

毒舌但有用的 AI 助手 - iOS/macOS 客户端

## 功能特性

- 🎤 **语音对话** - 支持语音输入和语音播报
- 💬 **多模型对话** - 支持 GPT-4、Claude、Gemini 等多种模型
- 💡 **毒舌人设** - AI 用独特的方式帮助你
- ⭐ **收藏功能** - 收藏喜欢的金句和对话
- 📊 **数据统计** - 查看使用统计和分析

## 技术栈

- Swift 5.9+
- SwiftUI
- iOS 16+ / macOS 13+
- AVFoundation (语音合成/识别)

## 项目结构

```
Caobao/
├── CaobaoApp.swift          # 应用入口
├── Models/                   # 数据模型
├── Views/                    # 视图层
├── ViewModels/               # 视图模型
├── Services/                 # 服务层
└── DesignSystem/             # 设计系统
```

## 开发环境

- Xcode 15+
- iOS 16.0+ / macOS 13.0+

## 配置

1. 打开 `Caobao.xcodeproj`
2. 修改 `APIService.swift` 中的 API 地址
3. 运行项目
